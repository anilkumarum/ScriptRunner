How to use scriptRunner

Pin scriptRunner after install extension
Click on scriptRunner icon or (Alt + S ) to open scriptRunner popup Window

ℹ️ Note

Need to enable developer mode at chrome://extensions

Add a script

Right click on scriptRunner icon and then select Script Manager options from popup menu

  1. Create Script
    Click on Plus floating button on bottom right corner of script Manager
    then click on Add script sub floating button.
    Form dialog open. Fill form then click on Submit button to Add Script
  2. Add local Script file
    Click on Plus floating button on bottom right corner of script Manager
    then click on Open file sub floating button.
    Pick mutliple .js files from local disk then edit script form.
    Finally, click on Submit button to Add Script.
    When you edit local file in any ide or text editor, scriptRunner auto-update script.

View error or debug logs in scriptRunner console

If your script don't work properly, you can check error thrown by browser
Open scriptRunner popup Window
Click on Console icon on bottom right side of script item.
Console dialog will open
